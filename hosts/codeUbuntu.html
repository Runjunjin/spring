Ubuntu20.04安装教程

发布时间2022/3/5

1、安装VMware

2、下载Ubuntu 20.04映像文件

3、创建自定义虚拟机，联网（桥接式），内存2G，CPU核数1（内核数3），磁盘空间划分20G

4、等待安装结束

5、打开terminal

输入更新命令：

sudo apt-get update

sudo apt-get upgrade

6、安装中文输入法：打开language support

选择install/remove language

选择Chinese simplifie

打开terminal输入：

sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4回车

im-config -s ibus或者im-switch -s ibus

sudo apt-get install language-pack

sudo apt install ibus-libpinyin

sudo apt install ibus-clutter

sudo reboot

打开settings找到region&language添加Chinese intellegent pinyin

在terminal中输入：ibus-setup

在input method中添加即可

7、通过Ubuntu商店下载pycharm安装完毕后打开

8、help中edit custom VM options...

9、编辑以下代码：

-Drecreate.x11.input.method=ture
